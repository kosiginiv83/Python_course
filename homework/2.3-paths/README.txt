find_procedure.py - Поиск только по названиям файлов.

find_procedure_2.py - Поиск только по названиям файлов. !!! Внимание: find_procedure_2.py ищется по всему диску, ссылка на который в переменной окружения ОС 'HOMEDRIVE'. find_procedure_2.py должен быть один на диске.
